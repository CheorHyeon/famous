## 프로젝트 개요
- 멋쟁이 사자처럼 백엔드스쿨 3기 - Java 수업 이후 기능 구현 미션을 위한 프로젝트
- Java를 활용한 명언 생성, 삭제, 조회, 수정 기능

## 각 단계별 설명
- 1단계 : 콘솔창에서 "종료" 명령어 입력 시 프로그램 종료
- 2단계 : "등록" 명령어 시 명언 등록
- 3단계 : "등록" 명령어 시 명언번호 노출
- 4단계 : 등록할때 마다 생성되는 명언번호 증가
- 5단계 : "목록" 등록된 명언 조회
- 6단계 : 명언 삭제
- 7단계 : 존재하지 않은 명언 삭제 시 예외처리
- 8단계 : 명언 수정
